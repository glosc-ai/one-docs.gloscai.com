# Open Design - 开源的 UI 设计工具

## 概述

Open Design 是官方、本地优先的 Claude Design 替代方案。你现有的编码 Agent —— Claude Code · Codex · Cursor · Gemini · OpenCode · Qwen —— 会变成设计引擎，并由 132 个可组合 Skill 与 150 套可移植 DESIGN.md 系统驱动。

## 官方资源
- GitHub：https://github.com/nexu-io/open-design
- 官网: https://open-design.ai/zh/

## 安装

在 https://github.com/nexu-io/open-design/releases 下载最新版本的 Open Design，解压后运行安装包进行安装。


## 配置方法

方式选择 "自己的模型key"

提供方选择 "OpenAI" 自定义提供方

Base URL 填写 `https://one.gloscai.com/v1`
API Key 填写你在 [key](https://one.gloscai.com/keys) 获取的密钥

点击 拉取模型，然后在模型中选择你想用的模型，点继续
