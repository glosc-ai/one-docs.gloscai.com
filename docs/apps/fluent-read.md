# 流畅阅读 (FluentRead) - 开源翻译插件



## 概述

流畅阅读 FluentRead 是开源浏览器翻译插件，支持传统翻译服务和 AI 大模型翻译，可通过 New API 接入自定义模型，实现沉浸式双语阅读体验。

## 官方资源

- GitHub：<https://github.com/Bistutu/FluentRead>

## 核心能力

| 能力         | 说明                                                                                     |
| ------------ | ---------------------------------------------------------------------------------------- |
| 智能翻译引擎 | 支持 20 多种翻译引擎，包括微软翻译、谷歌翻译、DeepL、OpenAI、DeepSeek、Kimi、Ollama 等。 |
| 沉浸式阅读   | 支持双语对照、划词翻译、一键复制和全文翻译。                                             |
| 隐私与定制   | 数据本地存储，代码开源透明，提供较丰富的自定义选项。                                     |

## New API 控制台快捷项

可在 New API 控制台的系统设置、聊天设置中添加快捷选项：

```json
{
  "流畅阅读": "fluentread"
}
```

## 配置方法

### 从 New API 控制台导入配置

1. 安装流畅阅读浏览器插件。
2. 打开 New API 控制台的令牌管理页面。
3. 页面提示添加流畅阅读时，选择模型并点击一键填充到 FluentRead。
4. 在确认窗口中检查 API 地址、令牌和模型信息。
5. 确认导入后，流畅阅读中的 New API 配置会启用。

### 在流畅阅读中手动填写配置

| 字段         | 填写方式                                            |
| ------------ | --------------------------------------------------- |
| 翻译服务     | 选择 New API。                                      |
| 访问令牌     | 填写 New API 密钥。                                 |
| New API 接口 | 填写 New API 部署地址，官方页面提示此处不带 `/v1`。 |
| 模型         | 从列表中选择，或使用自定义模型。                    |
| 自定义模型   | 填写 New API 中暴露的模型名称。                     |
