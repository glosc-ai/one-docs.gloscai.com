## 快速开始

除特别说明外，请求需要携带 Bearer Token，并按接口类型设置请求体格式。

```http
Authorization: Bearer sk-xxxxxx
Content-Type: application/json
```

```bash
curl -X GET "https://one.gloscai.com/v1/models" \
  -H "Authorization: Bearer sk-xxxxxx"
```
